Python is a useful programming language. When developing NES games, I find it useful to quickly write repetive bits of code or do calculations in hexadecimal.

One simple script can turn X and Y coordinates into a nametable address, which can be useful for drawing graphics to the screen:





Nametable A: (0x2000|(((y)<<5)|(x)))
Nametable B: (0x2400|(((y)<<5)|(x)))
Nametable C: (0x2800|(((y)<<5)|(x)))
Nametable D: (0x2c00|(((y)<<5)|(x)))
