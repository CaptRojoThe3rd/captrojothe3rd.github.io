A hex editor will be useful for creating CHR files of a specific size, especially since YYCHR creates 16 KiB / 4 page CHR files by default.

To create an empty CHR file, just create a new file in your hex editor and fill it with zeroes. A basic NES CHR-ROM should be 8 KiB (0x2000) in size.
