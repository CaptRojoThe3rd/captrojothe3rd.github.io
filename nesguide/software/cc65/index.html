Download CC65 here

CC65 is a compiler suite for the 6502, the processor that the NES's processor (the 2A03) is based on.

When you download it, you can choose where you want to install it. If you are on Windows, I recommend just putting it in C:\ or in C:\Users\<yourname>.
You should also add it to PATH/your enviroment variables.

CC65 contains CA65, which is an assembler for the 6502. It is what will turn the code you write into an NES ROM file that an emulator can understand.
