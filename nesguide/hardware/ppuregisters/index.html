The PPU has registers available to the CPU located at $2000-$2007 (mirrored at $2008-$3fff). There are 8 registers, although they serve many more than 8 functions. $2000 and $2001 have many 1-bit flags.


$2000: PPU Control
$2001: PPU Mask
$2002: PPU Status
$2003: OAM Address
$2004: OAM Data
$2005: PPU Scroll
$2006: PPU Address
$2007: PPU Data



The following are the descriptions of the PPU's registers. Some registers have different effects depending on if the write is the 1st or 2nd write in a pair.


$2000: PPU Control

VPHB SINN
|||| ||++-- Base nametable address (0 = $2000; 1 = $2400; 2 = $2800; 3 = $2c00)
|||| |+---- VRAM address increment per CPU read/write of PPU Data (0: add 1, going across; 1: add 32, going down)
|||| +----- Sprite pattern table address for 8x8 sprites (0: $0000; 1: $1000)
|||+------- Background pattern table address (0: $1000; 1: $1000)
||+-------- Sprite size (0: 8x8; 1: 8x16)
|+--------- PPU master/slave select (0: read backdrop from EXT pins; 1: output color on EXT pins)
+---------- Generate an NMI at the start of the vertical blanking interval (0: off; 1: on)


$2001: PPU Mask

BGRs bMmG
|||| |||+-- Display type: (0: color, 1: grayscale)
|||| ||+--- 1: Show background in leftmost 8 pixels of screen, 0: Hide
|||| |+---- 1: Show sprites in leftmost 8 pixels of screen, 0: Hide
|||| +----- 1: Show background
|||+------- 1: Show sprites
||+-------- Emphasize red
|+--------- Emphasize green
+---------- Emphasize blue


$2002: PPU Status

VSO. ....
|||+-++++-- Least significant bits previously written into a PPU register
||+-------- Sprite overflow (bugged)
|+--------- Sprite 0 Hit
+---------- vBlank (0: not in vBlank; 1: in vBlank)


$2003: OAM Address

Set OAM Address


$2004: OAM Data

Read/Write from OAM RAM


$2005: PPU Scroll

Set PPU Scroll. 1st Write = X Scroll, 2nd Write = Y Scroll


$2006: PPU Address

Set PPU Address. 1st Write = High Byte, 2nd Write = Low Byte


$2007: PPU Data

Read/Write from VRAM
