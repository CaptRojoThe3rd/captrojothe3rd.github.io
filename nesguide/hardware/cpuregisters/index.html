The 2a03 has many registers other than the 6502 registers. They can be accessed at $4000-$401f. Many of these registers are for the APU, or Audo Processing Unit.


$4000-$4003: Square Channel 0
$4004-$4007: Square Channel 1
$4008-$400b: Triangle Channel
$400c-$400f: Noise Channel
$4010-$4013: DPCM Channel
$4014-$----: OAM DMA
$4015-$----: APU Status
$4016-$----: Controller Strobe / Controller 1 Data
$4017-$----: APU Frame Counter / Controller 2 Data
$4018-$401f: APU and I/O functionality that is normally disabled (https://www.nesdev.org/wiki/CPU_Test_Mode)



For descriptions of APU/audio registers, see the appropriate sections of this guide. Here are the descriptions of non-APU registers:


$4014: OAM DMA / Sprite DMA

Writing XX to this register will write the data at $xx00-$xxff to PPU register $2004.


$4016: Controller Strobe / Controller 1 Data

Writing $01 followed by $00 to this register will begin the process of controller reading.

Reading from this register will return Controller Port 1 data.


$4017: APU Frame Counter / Controller 2 Data

Write: APU Frame Counter

MI-- ----

|+-------- IRQ inhibit flag

+--------- Mode (0 = 4-step, 1 = 5-step)

Reading from this register will return Controller Port 2 data.
